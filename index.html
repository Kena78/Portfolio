Сайт для создание блок-схем в соответвии с ГОСТ

На данном сайте релизована генерация блок-схем.

Блок-схемы описываются с помощью псевдо языка

Имеется:

  • Подсветка синтаксиса
  • Автодополненение закрывающих элементов(скобки и кавычки)
  • Ветвления в боковые стороны

Intellij Idea

Intellij Idea — Среда для разработки на Java. На её основе так же сущестуют среды разработки для других языков(PhpStorm, PyCharm, GoLang и тд)

ArcPlugin

Этот плагин добавляет поддержку фреймворка Arc в Intellij Idea

  • Специально отображение некоторые классов в debug режиме
  • Отображение цветовых преобразований на боковой понели
  • Инспекции предупреждающие о Runtime ошибка
  • Автодополнение из .properties файлов, в соответсвии с конвенциями Arc
  • Шаблоны кода для колекций из Arc

HJson

Для языка hjson добавляет подсветку синтаксиса, автоформатирование, и остальные элементы поддержки языка в среду разработки IntellijIdea.
